使用Docker搭建Ngrok服务器

发表于 2017-12-09 | 分类于 ngrok | 阅读次数

使用Docker搭建Ngrok服务器

docker pull hteen/ngrok

启动Ngrok serve

root@iZ25f738hs2Z ngrok]# docker run -idt --name ngrok-server \
-v /data/ngrok:/myfiles \
-p 80:80 \
-p 443:443 \
-p 4443:4443 \
-e DOMAIN='tunnel.hteen.cn' hteen/ngrok /bin/sh /server.sh

客户端连接

首先创建一个ngrok.cfg配置文件

server_addr: "tunnel.hteen.cn:4443"
trust_host_root_certs: false

然后在命令行执行

./ngrok -config ./ngrok.cfg -subdomain wechat 192.168.99.100:80

Nginx + Docker + Ngrok

server {
 listen       80;
 server_name  tunnel.hteen.cn *.tunnel.hteen.cn;
 location / {
         proxy_redirect off;
         proxy_set_header Host $host;
         proxy_set_header X-Real-IP $remote_addr;
         proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
         proxy_pass http://localhost:8082;
 }
 }
 server {
     listen       443;
     server_name  tunnel.hteen.cn *.tunnel.hteen.cn;
     location / {
             proxy_redirect off;
             proxy_set_header Host $host;
             proxy_set_header X-Real-IP $remote_addr;
             proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
             proxy_pass http://localhost:4432;
     }
 }
