Docker使用手册

Docker简介

Docker 是一个开源的应用容器引擎，基于 Go 语言 并遵从Apache2.0协议开源。

Docker 可以让开发者打包他们的应用以及依赖包到一个轻量级、可移植的容器中，然后发布到任何流行的 Linux 机器上，也可以实现虚拟化。

容器是完全使用沙箱机制，相互之间不会有任何接口（类似 iPhone 的 app）,更重要的是容器性能开销极低。

微道云的构建服务器Jenkins环境介绍

  1. 服务器安装git yum install git
  2. 下载Jenkins，上传到服务器
  3. 登陆服务器运行Jenkins java -jar jenkins.war --ajp13Port=-1 --httpPort=8081，其中8081就是新的端口号，也可以设置为其他的端口
  4. Jenkins新增插件Gitlab 和 Git Parameter。插件安装在系统管理 -> 管理插件菜单中

如何设置阿里云Docker加速

修改/etc/docker/daemon.json 文件 (centos7系统），如无新增。

"registry-mirrors": ["https://2l5irm5o.mirror.aliyuncs.com"]

https://2l5irm5o.mirror.aliyuncs.com这个地址需要自己去阿里云注册个账号后，进入自己的控制台，获取国内加速地址。

ECS通过内网推送镜像

每个阿里云账户的username不一样，每个区的registry地址不一样，请根据情况填写相关地址

如何登陆阿里云docker registry:

$ sudo docker login --username=嗯嗯eeee registry.cn-hangzhou.aliyuncs.com

登录registry的用户名是您的阿里云账号全名，密码是您开通服务时设置的密码。

你可以在镜像管理首页点击右上角按钮修改docker login密码。
