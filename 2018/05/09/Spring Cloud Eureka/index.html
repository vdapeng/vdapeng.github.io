Spring Cloud Eureka

常用设置

手工指定客户端IP，添加以下配置：

# 指定此实例的ip
eureka.instance.ip-address=192.168.0.1
# 注册时使用ip而不是主机名
eureka.instance.prefer-ip-address=true

实用security进行注册身份验证

服务端添加如下配置：

security:
  basic:
    enabled: true
  user:
    name: admin
    password: admin
    role:
    - USER
    - ADMIN

客户端注册时，需配置用户名和密码，如下配置：

eureka:
    client:
        serviceUrl: 
          defaultZone: http://admin:admin@192.168.11.136:7070/eureka/
